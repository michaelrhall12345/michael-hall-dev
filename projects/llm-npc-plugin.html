// systems.overview
01
GPT API Integration
Async HTTP requests to the OpenAI API from C++. Responses streamed back and injected into the NPC dialogue pipeline at runtime.
02
Personality Data Assets
Per-character personality profiles defined in UE5 Data Assets. Designers tune tone, knowledge scope, and behavioral constraints without touching code.
03
Context Window Management
Conversation history tracked per NPC instance. Context windows managed to stay within API token limits while preserving dialogue continuity.
04
Blueprint-Accessible Interface
Full plugin interface exposed to Blueprint so designers can trigger dialogue, pass context, and handle responses without writing C++.
